## 📋 Overview

This repository contains the complete source code for my personal developer portfolio — a hand-crafted, framework-free, single-page application built to present my identity as a developer, demonstrate a breadth of real-world projects, and provide a direct channel for professional contact.

The design prioritises clarity, performance, and user experience. Every section has been deliberately structured to communicate technical competence and professional maturity to both technical peers and non-technical stakeholders.

> **Design Philosophy:** No frameworks. No bloat. Clean, maintainable, and fully dependency-light frontend development using only the core technologies of the web.

## ⚙️ Getting Started

This project requires no build tools, package managers, or runtime dependencies.

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, or Safari)
- Optionally: [VS Code](https://code.visualstudio.com/) with the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension

### Installation

**Step 1 — Clone the repository**
```bash
git clone https://github.com/AliAbbasCTN/portfolio.git
cd portfolio
```

**Step 2 — Launch the project**

Option A — Open directly in a browser:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Option B — Use a local development server *(recommended to avoid CORS issues with the PDF viewer)*:
```bash
# Python 3
python -m http.server 8000
# Then visit: http://localhost:8000
```

Option C — VS Code Live Server:
> Right-click `index.html` in the Explorer panel → **Open with Live Server**

## 🎨 Customization Guide

To adapt this portfolio for your own use, the following modifications are recommended:

| What to Change | Location | Instructions |
|---|---|---|
| Name & bio | `index.html` → sidebar & `#about` | Update the `<h2>` and `<p>` tags |
| Profile photo | `Ali7.png` | Replace the file; update the `src` attribute in `index.html` |
| Skill names & levels | `index.html` → `.skills-grid` | Edit `.skill-name`, `.skill-level`, and the inline `width` on `.skill-progress` |
| Projects | `index.html` → `.projects-grid` | Add, remove, or edit `.project-card` blocks and their `data-category` attributes |
| Resume | `Ali Resume.pdf` | Replace with your own PDF; update `href` and `src` references in `index.html` |
| Theme colours | `style.css` | Modify the CSS custom properties in the `:root` selector |
| Social links | `index.html` → `.socials` and `footer` | Update the `href` values on each anchor element |
| Contact form backend | `script.js` | Integrate [Formspree](https://formspree.io/) or [EmailJS](https://www.emailjs.com/) for live form submission |
| Favicon | `A.png` | Replace the file; update the `<link rel="icon">` tag in `<head>` |
